# قواعد داده ربات تعطیلات کوروش‌یار v1.2.5

1. تعطیلات سالانه فقط از تقویم رسمی کشور وارد می‌شوند.
2. تعطیلی‌های موردی فقط با اعلام صریح مرجع رسمی منتشر می‌شوند.
3. عزای عمومی به‌تنهایی تعطیلی محسوب نمی‌شود.
4. تعطیلی شهر یا شهرستان به کل استان تعمیم داده نمی‌شود.
5. ساعت کاری چندماهه یک رکورد بازه‌ای با `date` و `endDate` است.
6. پایان زودهنگام، شروع با تأخیر و دورکاری موردی رکورد مستقل روزانه هستند.
7. ساعت کاری دوره‌ای هیچ مهلت یا تاریخ پرونده‌ای را تغییر نمی‌دهد.
8. داده‌ها پیش از انتشار امضا و اعتبارسنجی می‌شوند.
