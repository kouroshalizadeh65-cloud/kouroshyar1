# کوروش‌یار v3.6.59+131

سورس کامل Flutter/Android نسخه `3.6.59+131` است و از نسخه نصب‌شده و رسمی `3.6.58+130` ادامه یافته است.

## تغییر اصلی

تقویم آنلاین اکنون تصمیم‌های رسمی تعطیلی، دورکاری و تغییر ساعت کاری را در سه سطح سراسری، استانی و شهرستانی می‌پذیرد. با انتخاب یک استان، تصمیم‌های مربوط به همه شهرستان‌های همان استان نیز نمایش داده می‌شوند؛ ولی تعطیلی شهرستانی به‌عنوان تعطیلی کل استان علامت‌گذاری نمی‌شود.

ساختار خوراک از این فیلدها پشتیبانی می‌کند:

- `scope: national` برای سراسر کشور
- `scope: province` برای کل استان
- `scope: county` همراه با `counties` برای یک یا چند شهرستان
- `excludedCounties` برای تصمیم‌هایی مانند «سایر شهرستان‌های استان به‌جز ...»

## ربات

ربات یکپارچه `1.4.0` علاوه بر منابع تخصصی و ۳۱ خوراک استانی ایرنا، کانال عمومی رسمی استانداری ایلام و یک لایه جستجوی تکمیلی برای هر ۳۱ استان را بررسی می‌کند. بازه ۱۴ روز گذشته در هر اجرا دوباره پایش می‌شود و تغییر متن یک اطلاعیه در همان نشانی با هش محتوا تشخیص داده می‌شود. اگر منبع حیاتی از دسترس خارج شود یا پوشش منابع از حد ایمن پایین‌تر بیاید، Workflow قرمز می‌شود و خوراک ناقص منتشر نمی‌شود. فقط اطلاعیه دارای تاریخ، محدوده و انتساب رسمی روشن منتشر می‌شود. کف بازبینی زنده موجود در مخزن حفظ می‌شود و فقط خوراکی افزایش می‌یابد که محتوای آن واقعاً تغییر کرده باشد.

## کنترل ساخت

GitHub Actions به‌ترتیب `flutter pub get`، `flutter test`، `flutter analyze` و `flutter build apk --release` را اجرا می‌کند. سپس موارد زیر کنترل می‌شوند:

- Package ID: `com.example.kouroshyar`
- تطابق `versionName` و `versionCode` با `pubspec.yaml`
- امضای APK و در صورت تنظیم، اثرانگشت گواهی
- نام خودکار APK و Artifact بر اساس نسخه واقعی

keystore، رمزهای امضا، کلید خصوصی خوراک، `key.properties`، APK و `pubspec.lock` داخل بسته سورس قرار ندارند.
