- تقویم داخلی و تعطیلات رسمی سراسری به‌صورت آفلاین
- بدون ربات، خوراک اینترنتی، تعطیلات استانی و تغییر ساعات کاری
- بدون مجوز اینترنت در AndroidManifest
- بدون فایل کلید، keystore یا رمز امضا در سورس

## کنترل ساخت

گردش‌کار GitHub Actions به‌ترتیب مراحل زیر را اجرا می‌کند:

```text
flutter pub get
flutter test
flutter analyze
flutter build apk --release
```

پس از ساخت، امضای APK و شناسه بسته نیز بررسی می‌شوند. ساخت Release فقط با Secrets امضای دائمی مخزن انجام می‌شود.
