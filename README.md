# کوروش‌یار v3.6.55+127

سورس کامل Flutter/Android نسخه `3.6.55+127` است و مستقیماً از نسخه رسمی `v3.6.54+126` ادامه یافته است.

تغییرات این نسخه:

- ساعت کاری چندماهه به‌صورت متن ساده زیر نمای ماهانه نمایش داده می‌شود.
- همان متن در جزئیات روزهای داخل بازه نمایش داده می‌شود.
- ساعت کاری دوره‌ای علامت ساعت، رنگ هشدار یا اعلان روزانه ندارد.
- تغییرات موردی مانند پایان زودهنگام، شروع با تأخیر یا دورکاری همچنان با نشان جداگانه نمایش داده می‌شوند.
- تعطیلی‌های رسمی سراسری ۱۴ و ۱۵ تیر ۱۴۰۵ به داده آفلاین افزوده شده‌اند.
- دریافت تعطیلات و ساعات کاری امضاشده، تقویم شمسی آفلاین و همه قابلیت‌های امنیتی نسخه قبل حفظ شده‌اند.
- هیچ مهلت، کار، جلسه یا تاریخ پرونده‌ای به‌صورت خودکار تغییر نمی‌کند.

## کنترل ساخت

گردش‌کار GitHub Actions به‌ترتیب این مراحل را اجرا می‌کند:

```text
flutter pub get
flutter test
flutter analyze
flutter build apk --release
```

پس از Build، امضای APK و شناسه بسته `com.example.kouroshyar` بررسی می‌شوند. keystore، رمزهای امضا و `key.properties` داخل سورس قرار ندارند.
