## محدودیت اعتبارسنجی

Flutter و Dart در محیط محلی این بسته نصب نبودند؛ بنابراین دستورات زیر در این محیط اجرا نشده‌اند:

```text
flutter pub get
flutter test
flutter analyze
flutter build apk --release
```

در نتیجه، موفقیت کامپایل یا Build در این سند ادعا نمی‌شود. GitHub Actions طوری تنظیم شده که در صورت شکست هر یک از تست، Analyze، Build، بررسی package ID یا امضا، خروجی APK منتشر نشود.

## pubspec.lock

فایل `pubspec.lock` به‌صورت دستی یا حدسی ساخته نشده است. تولید معتبر آن نیازمند اجرای واقعی `flutter pub get` و حل کامل وابستگی‌هاست. سه وابستگی جدید اعلان و منطقه زمانی در `pubspec.yaml` با نسخه دقیق تثبیت شده‌اند تا دامنه تغییرات resolution محدود شود.
