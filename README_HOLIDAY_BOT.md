# ربات تعطیلات و ساعات کاری کوروش‌یار v1.1.1

این نسخه فقط یک Workflow با نام `KouroshYar Holiday Bot` دارد.

## داده پایه

- ۲۶ تاریخ تعطیل رسمی تقویم سالانه ۱۴۰۵.
- تعطیلی رسمی موردی سراسر کشور در ۱۴ و ۱۵ تیر ۱۴۰۵.
- یک رکورد بازه‌ای ساعت کاری ۷:۰۰ تا ۱۳:۰۰ از ۲۶ اردیبهشت تا ۱۵ شهریور ۱۴۰۵.
- تغییر موردی ساعت کاری استان ایلام در ۲۴ تیر ۱۴۰۵.

## قواعد ایمنی

- تعطیلی شهرستانی به کل استان تعمیم داده نمی‌شود.
- عزای عمومی بدون اعلام صریح تعطیلی، تعطیلی محسوب نمی‌شود.
- خبر احتمالی، شایعه یا تکذیبی منتشر نمی‌شود.
- کلید خصوصی فقط از Secret با نام `HOLIDAY_FEED_PRIVATE_KEY_B64` خوانده می‌شود.
- هیچ کلید خصوصی، keystore یا `key.properties` در سورس قرار ندارد.

## اجرای دستی

در Actions، Workflow `KouroshYar Holiday Bot` را با تنظیم زیر اجرا کنید:

```text
mode = current
dry_run = false
```

نسخه طبقه‌بندی: `1.1.1-r1`.
