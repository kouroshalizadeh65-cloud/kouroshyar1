# KouroshYar v3.4.3

## تغییرات اصلی

- اصلاح صفحه پروفایل:
  - افزودن SafeArea و padding پایین برای جلوگیری از افتادن دکمه ذخیره زیر Navigation Bar.
  - جلوگیری از ذخیره پروفایل بدون نام کاربری.
  - نمایش پیام «حداقل نام کاربری را وارد کنید.» برای نام کاربری خالی.
  - حفظ اختیاری بودن عنوان، شماره پروانه و کانون/مرکز.

- تکمیل تایپ صوتی:
  - تفکیک پیام خطای مجوز میکروفون.
  - تفکیک پیام آماده نبودن سرویس گفتار.
  - تفکیک پیام نبود پشتیبانی فارسی.
  - حفظ راهکار جایگزین میکروفون صفحه‌کلید.

- تکمیل اتاق پرونده:
  - افزودن ویرایش و حذف برای کارهای پرونده.
  - افزودن ویرایش و حذف برای رویدادهای خط زمان پرونده.
  - حفظ نمایش تاریخ شمسی در کارها و رویدادها.
  - نمایش پیام واضح برای عنوان خالی کار و رویداد.

- اصلاح مهلت‌ها:
  - جلوگیری از ثبت مهلت بدون عنوان.
  - نمایش پیام «عنوان مهلت را وارد کنید.» برای عنوان خالی.
  - نمایش پیام «ثبت شد» پس از ثبت موفق.

## نسخه

- به‌روزرسانی نسخه از 3.4.2+52 به 3.4.3+53.
