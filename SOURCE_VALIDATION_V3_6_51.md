# گزارش اعتبارسنجی سورس v3.6.51+123

## کنترل‌های انجام‌شده در محیط تولید سورس

- سلامت ساختار ZIP مبنا و استخراج فقط از `v3.6.50+122`
- بررسی نحوی ساختاری ۶۵ فایل Dart شامل توازن پرانتز، کروشه، آکولاد، رشته و توضیحات
- بررسی تمام import/exportهای نسبی و نبود مسیر مفقود
- parse سه فایل YAML اصلی و ۱۱ فایل XML
- بررسی شناسه بسته Android و مسیر MainActivity
- بررسی وجود receiverها و مجوزهای اعلان Android
- بررسی نام نسخه در `pubspec.yaml`، رابط برنامه، manifest پشتیبان و GitHub Actions
- بررسی نبود keystore، `key.properties`، رمز، کلید خصوصی و symlink در خروجی
- بررسی سازگاری رمز حداقل ۶ نویسه برای پشتیبان دستی و حداقل ۴ رقم برای پشتیبان اضطراری بازیابی
- بررسی هشدار و تأیید جداگانه برای پشتیبان‌های تقریبی بزرگ‌تر از ۲۵۰ مگابایت
- بررسی سلامت ZIP نهایی و مسیرهای امن اعضای آن

## محدودیت اعتبارسنجی

Flutter و Dart در محیط محلی نصب نبودند؛ بنابراین این سه دستور در این محیط اجرا نشده‌اند:

```text
flutter test
flutter analyze
flutter build apk --release
```

نتیجه قطعی کامپایل، تست، امضا و ساخت APK فقط بعد از سبزشدن همه مراحل GitHub Actions مشخص می‌شود. نصب APK باید روی نسخه `3.6.50+122` آزمایش شود تا حفظ داده‌ها و سازگاری امضا نیز تأیید گردد.
