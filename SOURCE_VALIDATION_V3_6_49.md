## محدودیت اعتبارسنجی

Flutter و Dart در محیط محلی این بسته نصب نبودند؛ بنابراین دستورات زیر در این محیط اجرا نشده‌اند:

```text
flutter pub get
flutter test
flutter analyze
flutter build apk --release
```

در نتیجه، موفقیت کامپایل، Analyze یا Build در این سند ادعا نمی‌شود. GitHub Actions طوری تنظیم شده که در صورت شکست تست، Analyze، Build، بررسی شناسه بسته یا بررسی امضا، APK منتشر نشود.

## pubspec.lock

فایل `pubspec.lock` به‌صورت دستی یا حدسی ساخته نشده است. تولید معتبر آن نیازمند اجرای واقعی `flutter pub get` است. نسخه وابستگی‌های اعلان و منطقه زمانی در `pubspec.yaml` به‌صورت دقیق تثبیت شده‌اند تا دامنه تغییرات resolution محدود شود.
