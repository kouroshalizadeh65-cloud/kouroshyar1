# KouroshYar v3.4.6 — Today & Reports Global Dashboard

این نسخه برای نزدیک‌تر کردن کوروش‌یار به الگوی برنامه‌های حرفه‌ای مدیریت دفتر وکالت ساخته شد؛ تمرکز روی داشبورد امروز، گزارش‌های تصمیم‌ساز و کاهش نمایش‌های صرفاً آماری است.

## تغییرات اصلی

- تبدیل گزارش‌ها به صفحه «گزارش‌های تصمیم‌ساز».
- افزودن جمع‌بندی اولویت روز بر اساس مهلت‌ها، کارهای عقب‌افتاده، جلسات و پرونده‌های بی‌اقدام.
- افزودن کارت‌های شاخص برای کار باز، کار عقب‌افتاده، مهلت فوری، پرونده فعال، جلسه امروز و مانده مالی.
- تفکیک مهلت‌های منقضی‌شده، مهلت‌های امروز و مهلت‌های ۷ روز آینده.
- افزودن گزارش جلسات امروز، فردا و ۷ روز آینده بر اساس خط زمان پرونده.
- افزودن گزارش مالی شامل درآمد کل، هزینه کل، مانده کل، درآمد امروز و هزینه امروز.
- افزودن گزارش اسناد، اشخاص پرونده و رویدادهای خط زمان.
- تقویت صفحه امروز با داشبورد تصمیم روز و پیام اولویت پیشنهادی.
- اصلاحات ظاهری سبک در AppBar، ListTile و Divider برای یکدست‌تر شدن نما.
- به‌روزرسانی نسخه به 3.4.6+56.
- حفظ اصلاح دائمی signingConfig release.
- حفظ نام یکتای APK و Artifact.

## محدودیت

در محیط فعلی Flutter/Dart نصب نیست؛ بنابراین Build واقعی، APK و تست گوشی انجام نشده است. نسخه تا زمان موفقیت GitHub Actions و تست روی گوشی، آماده نهایی محسوب نمی‌شود.
