# تغییرات v3.6.27

## محور

اصلاح شرح پرونده و تبدیل خروجی آن از حالت فرمی به متن طبیعی حقوقی.

## تغییرات

- حذف عبارت‌هایی مثل «مشخصات پرونده شامل...» از موتور تولید شرح پرونده.
- بازنویسی منطق درج مشخصات پرونده تا شماره پرونده، شماره بایگانی، شعبه، مرجع، قاضی و سایر اطلاعات در جمله‌های طبیعی استفاده شوند.
- افزودن حالت‌های «خلاصه کوتاه»، «شرح معمولی» و «شرح کامل» در بخش شرح پرونده.
- قابل ویرایش شدن متن تولیدی داخل کادر متنی.
- کپی کردن متن ویرایش‌شده به‌جای متن خام تولیدی.
- حفظ منطق استفاده از تاریخچه منتخب، مرتب‌سازی از قدیمی‌تر به جدیدتر و گزینه «ثبت در شرح پرونده».
- حفظ نگارش «رای» بدون همزه در متن‌های مربوط به تاریخچه و شرح پرونده.

## نکته build

در محیط ChatGPT ابزار Flutter/Dart نصب نبود؛ بنابراین `flutter analyze` و build محلی اجرا نشد و تست نهایی باید با GitHub Actions انجام شود.
