# KouroshYar v3.4.1

اصلاحات این نسخه:

- پیام آغازین پویا بر اساس نام کاربری؛ در صورت نبود نام، پیام عمومی نمایش داده می‌شود.
- افزودن صفحه ثبت‌نام / پروفایل کاربر.
- افزودن گزینه‌های استفاده یا عدم استفاده از نام، شماره پروانه و کانون/مرکز در متون حقوقی.
- اصلاح فرم امنیت: افزودن تکرار رمز، بررسی تطابق، حداقل ۴ رقم، و پاک شدن کادرهای رمز بعد از ذخیره.
- جلوگیری از درخواست قفل برنامه هنگام زدن تایپ صوتی؛ قفل فقط پس از بازگشت از پس‌زمینه و گذشت زمان مشخص فعال می‌شود.
- بهبود پیام خطای تایپ صوتی و انتخاب زبان فارسی در صورت پشتیبانی گوشی.
- پاک شدن متن فرمان/ثبت پس از ثبت موفق کار، مهلت یا مالی در بخش کوروش.
- افزایش نسخه به 3.4.1+51.
- اصلاح workflow ساخت APK و حذف اسکریپت‌های ناموجود.
- افزودن امضای ثابت برای Release APK جهت امکان آپدیت نسخه‌های بعدی روی هم.

نکته: اگر نسخه‌ای که قبلاً روی گوشی نصب شده با امضای متفاوت ساخته شده باشد، ممکن است فقط برای مهاجرت به امضای ثابت، یک‌بار حذف نسخه قبلی لازم شود. پس از نصب نسخه امضاشده ثابت، نسخه‌های بعدی باید به‌صورت بروزرسانی نصب شوند.
