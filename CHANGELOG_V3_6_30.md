# کوروش‌یار v3.6.30+102

## محور نسخه
ادغام مدارک پرونده با تاریخچه و مالی، حذف بخش مستقل مدارک پرونده، و افزودن نمایشگر داخلی فایل‌ها.

## تغییرات اصلی
- حذف کارت مستقل «مدارک پرونده» از اتاق پرونده.
- افزودن امکان پیوست مدرک به هر سابقه تاریخچه پرونده.
- افزودن امکان پیوست مدرک مالی/رسید به هر ثبت مالی پرونده.
- کپی فایل انتخاب‌شده در حافظه اختصاصی برنامه بدون فشرده‌سازی یا تبدیل فرمت.
- نمایش داخلی PDF داخل کوروش‌یار با پیمایش صفحه، زوم و نمایش شماره صفحه.
- افزودن جستجو و استخراج متن برای PDFهای متنی.
- نمایش داخلی عکس با زوم دو انگشتی، حرکت روی تصویر و چرخش.
- افزودن اشتراک‌گذاری فایل اصلی ذخیره‌شده.
- برای فایل‌های ناشناخته، پیشنهاد باز کردن با برنامه‌های دیگر گوشی.
- اصلاح مسیر ذخیره فایل‌ها به `case_attachments/case_<id>` برای جلوگیری از وابستگی به مسیر اصلی گوشی.

## نکات تست
در محیط ChatGPT، Flutter/Dart نصب نبود و `flutter analyze` یا build محلی اجرا نشد. تست نهایی باید با GitHub Actions انجام شود.
