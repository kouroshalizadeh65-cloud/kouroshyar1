# KouroshYar v3.6.0 - Case File Redesign

- بازطراحی ساختار صفحه داخلی پرونده‌ها با تمرکز بر سه بخش اصلی: مشخصات پرونده، مالی پرونده و مدارک پرونده.
- ادغام نمایش مشخصات، خلاصه، اشخاص، تاریخچه و اقدامات در بخش «مشخصات پرونده» به‌صورت تمیز و بازشونده.
- جدا شدن کامل بخش مالی پرونده از صفحه خانه؛ مالی‌های پرونده دیگر در فهرست امروز نمایش داده نمی‌شوند.
- بازچینی مالی پرونده به شکل یک بخش مستقل با خلاصه حق‌الوکاله، دریافتی، مانده، هزینه‌ها و عملیات ثبت/ویرایش/حذف.
- ساخت بخش مستقل «مدارک پرونده» با منوی سه‌نقطه، افزودن فایل، گرفتن عکس و ثبت سند بدون فایل.
- جستجو/ورود مستقیم نوع مدرک در کادر نوع مدرک با امکان ثبت نوع مدرک جدید.
- افزودن گزینه‌های «ثبت و ذخیره» و «ذخیره و افزودن بعدی» هنگام ثبت مدرک.
- حذف نمایش‌های تکراری و شلوغ از صفحه اول پرونده؛ صفحه ورودی پرونده فقط سه کارت اصلی را نشان می‌دهد.
- به‌روزرسانی نسخه به 3.6.0+72 و نام APK نسخه‌دار.
